⚠️ REQUIREMENTS:

✅ Backend must be running on port 8000
✅ Camera must be available (not in use)
✅ pose_landmarker_lite.task file in server/assets/
✅ Good lighting for pose detection
✅ Full upper body visible in frame


🐛 TROUBLESHOOTING:

❌ No skeleton lines?
   → Check backend terminal for MediaPipe errors
   → Ensure good lighting
   → Stand further from camera

❌ WebSocket error?
   → Verify backend is running
   → Check firewall settings
   → Look for errors in browser console

❌ Angle not detecting?
   → Make sure full upper body is visible
   → Face camera directly
   → Try better lighting
